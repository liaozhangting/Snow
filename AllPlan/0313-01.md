\### 1. 基础依赖安装 (耗时预估：20min)



首先需要安装 Java（Kafka 依赖）和 Mosquitto（MQTT 协议实现）。



```bash

\# 更新系统包列表

sudo apt update



\# 安装 OpenJDK 11 (Kafka 必须)

sudo apt install -y openjdk-11-jdk



\# 安装 Mosquitto (MQTT Broker) 及其客户端工具

sudo apt install -y mosquitto mosquitto-clients



\# 安装 Go 语言环境 (用于运行你的后端程序)

sudo apt install -y golang-go



\# 验证安装

java -version

mosquitto -v | head -n 1

go version



```



---



\### 2. Kafka 二进制包部署 (耗时预估：30min)



直接使用官网编译好的二进制文件，跳过复杂的编译过程。



```bash

\# 下载 Kafka 3.6.1 (以 3.6.1 为例)

wget https://archive.apache.org/dist/kafka/3.6.1/kafka\_2.13-3.6.1.tgz



\# 解压并重命名

tar -xzf kafka\_2.13-3.6.1.tgz

mv kafka\_2.13-3.6.1 /opt/kafka



\# 创建数据存放目录

mkdir -p /opt/kafka/data/zookeeper

mkdir -p /opt/kafka/data/kafka



```



---



\### 3. 内存极限调优配置 (最关键步骤)



\*\*必须手动限制 JVM 堆内存\*\*，否则默认配置会吃掉机器所有内存。



\#### A. 修改 Zookeeper 配置



编辑 `/opt/kafka/config/zookeeper.properties`：



\* 修改数据目录：`dataDir=/opt/kafka/data/zookeeper`



\#### B. 修改 Kafka 配置



编辑 `/opt/kafka/config/server.properties`：



\* 修改数据目录：`log.dirs=/opt/kafka/data/kafka`

\* 设置监听地址：`listeners=PLAINTEXT://0.0.0.0:9092`

\* \*\*重要：\*\* 如果要公网访问，设置 `advertised.listeners=PLAINTEXT://\[你的腾讯云公网IP]:9092`



---



\### 4. 启动与验证 (耗时预估：10min)



让雪毛儿按照顺序启动服务，并记录内存占用。



```bash

\# 1. 启动 Zookeeper (设置较小内存)

export KAFKA\_HEAP\_OPTS="-Xmx128M -Xms128M"

nohup /opt/kafka/bin/zookeeper-server-start.sh /opt/kafka/config/zookeeper.properties > /opt/kafka/zookeeper.log 2>\&1 \&



\# 2. 启动 Kafka (设置核心内存)

export KAFKA\_HEAP\_OPTS="-Xmx512M -Xms512M"

nohup /opt/kafka/bin/kafka-server-start.sh /opt/kafka/config/server.properties > /opt/kafka/kafka.log 2>\&1 \&



\# 3. 验证 MQTT

sudo systemctl start mosquitto



```

\### 5. 第一阶段验收标准



请雪毛儿在完成后截图发送以下信息：



1\. \*\*内存检查：\*\* 执行 `free -h`，确认 `used` 字段在 \*\*1.2G - 1.5G\*\* 之间。

2\. \*\*端口检查：\*\* 执行 `netstat -tunlp`，确认 `9092` (Kafka), `1883` (MQTT), `2181` (ZK) 都在监听。

3\. \*\*云端安全组：\*\* 确认腾讯云后台已开放 `9092` 和 `1883` 端口。
